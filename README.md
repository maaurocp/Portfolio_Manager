# Portfolio_Manager
Easy Python proyect using data from yfinance that gives insight on a chosen portfolio. It looks at a certain time horizon and shows basic return metrics, risk provided by each asset as well as a couple world scenarios involving more or less risk in the past. Most interestingly it uses Montecarlo to show what returns could be espected in the future 

Full Analysis Explained
1.Shows your Portfolio of choice
2.Basic past metrics:
  Annualized Returns
  Volatility
  Max drowdown
  CVar
3.Risk Contribution of each Ticker
4.Concentration index to evaluate diversifying
5. Scenario Comparison:
  Shows 3 possible past scenarios with more or less risk in the time frame
6.Monte Carlo Summary:
  Shows a future prediction on:
  -Mean
  -Median
  -Worst 5%
  -Best 95%
  -Probability to loose
